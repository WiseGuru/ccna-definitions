#### FDDI
- *Fiber Distributed Data Interface (FDDI)* is a network standard that uses Fiber connections in a LAN.
	- It is no longer commonly used, and has largely been replaced by [[CCNA/20 - Definitions/Ethernet\|Ethernet]]
