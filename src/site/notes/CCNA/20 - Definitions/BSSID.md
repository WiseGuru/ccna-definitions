- A *Basic Service Set Identifier* is a unique code that identifies an [[CCNA/20 - Definitions/Wireless Access Points\|AP]]'s wireless radio
	- Functionally it's a [[CCNA/20 - Definitions/MAC Address\|MAC Address]]
